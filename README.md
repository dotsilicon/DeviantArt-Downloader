# DeviantArt Image Downloader

A lightweight desktop app for collecting and downloading images from DeviantArt pages with less manual work. It helps extract image links and save artwork from supported pages through a simple desktop interface.

## Features

- Download images from DeviantArt pages
- Extract direct image URLs
- Simple desktop interface
- Fast workflow for repeated downloads
- Windows-friendly distribution

## Download

Get the latest version from the [Releases](../../releases) page.

## Installation

The app is distributed as a single Windows installer created with Inno Setup.

1. Download the latest installer from the **Releases** page
2. Run the installer
3. Follow the setup steps
4. Launch **DeviantArt Image Downloader** from the Start Menu or desktop shortcut

## How to Use

1. Open the app
2. Paste or enter the DeviantArt page URL
3. Start the scan or extraction process
4. Choose the images or links you want
5. Download and save them to your preferred folder

## System Requirements

- Windows 10 or later
- Internet connection

## Notes

- Download speed may depend on your network connection and the source page
- Availability of images may vary depending on page structure and access restrictions
- Please respect copyright, artist rights, and platform terms when downloading content
